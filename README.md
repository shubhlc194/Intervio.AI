## ✨ Features

- **AI-generated questions** — DSA, system design & behavioral questions tailored to your role
- **Smart feedback** — instant detailed feedback on your answers powered by Groq
- **Mock interview mode** — simulate real interview environments with timed sessions
- **Topic coverage** — Data Structures, Algorithms, System Design, STAR behavioral & more
- **Progress tracking** — monitor your performance over time

## 📦 Getting Started

### Prerequisites
- Node.js `v18+`
- A [Groq](https://console.groq.com) API key

### Installation

```bash
# 1. clone
git clone https://github.com/shubhlc194/InterviewIq.git
cd InterviewIq

# 2. install deps
npm install

# 3. set env vars
cp .env.example .env

# 4. fire it up
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

---

## ⚙️ Environment Variables

```env
VITE_GROQ_API_KEY=your_groq_api_key
```

> Never commit real keys — add `.env` to your `.gitignore`
